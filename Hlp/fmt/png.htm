.PNG - Portable network graphics

Features:

  • Pixel formats: Grayscale 1, 2, 4, 8, 16 bits/pixel. Palette 1, 2, 4, 8 bits/pixel. RGB 8, 16 bits/pixel. Uninterlaced and Adam7 interlaced supported.

  • Compression: Inflate / Deflate.

  • Transparency: Transparent color. Alpha plane.

  • Meta info: Name, Author, Comment, Copyright, Time, Source, Density, Creation date, Adobe XMP (Name, Author, Comment, Copyright, Keywords, Density, Creation date, GPS location).

  • Extensions: APNG - Animated PNG extension is supported.
