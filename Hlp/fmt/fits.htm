.FITS/.FIT - NOST Flexible image transport system

Features:

  • Pixel formats: 8, 16, 32 integer bits/pixel, 32, 64 float bits/pixel.

  • Compression: None.

  • Meta info: Name, Author, Comment, Creation date, Source.

  • Notes: This format is used by many astronomers, e.g. at NASA. 1D data is not supported. For data with dimension > 2, only the first 2D is used. The data 'range' is always mapped onto the full range of an 8-bit grayscale bitmap for display.
