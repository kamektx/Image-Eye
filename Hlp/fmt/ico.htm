.ICO - Microsoft Windows icon
.CUR - Microsoft Windows cursor

Features:

  • Pixel formats: 1, 4, 8, 24, 32 bits/pixel.

  • Compression: Uncompressed, PNG (deflate - used by Windows Vista and later).

  • Transparency: For 1, 4, 8 bits/pixel icons, the 'transparency mask' is translated into a transparent color. For 24 bits/pixel icons, the 'transparency mask' is translated into an alpha plane. For 32 bits/pixel icons (used by Windows XP and later), the icon contains an explicit alpha plane instead of mask.

  • Notes: Only the first (largest) mipmap is read.
