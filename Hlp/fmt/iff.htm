.IFF - Interchange file format
.BBM - Electronic Arts DeluxePaint brush
.LBM - Electronic Arts DeluxePaint bitmap

Features:

  • Pixel formats: 1..8, 24 bits/pixel planar type 49h and 8 bits/pixel type 50h, Amiga HAM6, HAM8 and Half-Brite images.

  • Compression: Uncompressed (type 0), Byte Run/RLE (type 1).

  • Transparency: Transparent color.

  • Meta info: Comment.

  • Note: IFF is a more general data storage format - only 'ILMB' image data is handled.
