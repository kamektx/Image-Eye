Slide show scripts

You can use Image Eye to display simple slide shows. The viewer and the index windows both have menu commands that help you create a slide show script. But you can also create and edit more advanced scripts 'by hand' using any text editor.

A slide show file is a text formatted file with an .IES extension. It must start with the text ImageEye SlideShow { followed by a variable number of statements and terminated with a }. A statement is a commands string followed by a variable number of argument strings terminated by a ;. A string is a sequence of characters terminated either by a white space or a ;. If you want to include white-space or ; in the string, enclose the whole string in a "-pair. To insert a " character in a string, use "". You can also use C++ style comments: /* ... */ for a multi-line comment or // for an 'until-the-end-of-the-line' comment.

Note: The text may be encoded either using the Windows 1252 code page (extended ISO 8859-1), or as UTF-8 encoded Unicode.

Here is an example named TEST.IES:

    /* 
 * This is an example slide show file
 */

ImageEye SlideShow {

  // First display a welcome picture and play a sound!
  display welcome.jpg; 
  shell "Play" "c:\windows\media\chord.wav";
  delay key;    // Wait for a keypress

  // Then display some product adds
  display prod1.bmp; delay 5;
  display "product 2.jpg"; delay 5;

  // Finally pop up a text file with further info
  shell open moreinfo.txt;

  // Toggle between two images until user press Esc
  loop {
    display "forever.png";
    display "and ever.png";
  }
}

These are the available commands:

    delay <n>Pause <n> seconds before continuing the slide show.
    delay keyWait for the user to press any key or left mouse button.
    display <str>Display the image file <str>.
    loop { ... }Loops the statements ... until Esc is pressed.
    loop <n> { ... }Loop <n> times the statements ...
    set <x> <n>Set option <x> to <n>.
    shell <str>Run the program file <str>.
    shell <verb> <str>Execute the Windows shell command <verb> on the document <str>.
Options for the set command are:

    AutoSizeImage on/offSet auto-sizing of image to video size (default: off).
Note: Don't forget to terminate each statement by a ;
