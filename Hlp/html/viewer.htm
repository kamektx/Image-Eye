Viewer commands

Right-button-click anywhere on an image viewer window to bring up this context menu:

Load new...Open a new image in the current window. [O, F3]
Spawn new...Open a new image file in a new window. [N, F4]

Next image fileDisplay the next image file in the folder. [PgDn, Space]
Previous image fileDisplay the previous image file in the folder. [PgUp, BackSpace]

File →
Load new...
Open a new image in the current window. [O, F3]
Next image file
 Display the next image file in the folder. [PgDn, Space]
Previous image file
Display the previous image file in the folder. [PgUp, BackSpace]
First image file in directory
Display the first image file in the folder. [Home]
Last image file in directory
Display the last image file in the folder. [End]
Close all image windows
Close all Image Eye windows. [c-a-X]

Spawn new...
Open a new image file in a new window. [N, F4]
Spawn next image
 Open the next image file in a new window. [c-PgDn, c-Space]
Spawn previous image
Open the previous image file in a new window. [c-PgUp, c-BackSpace]

Image file information...
View information about the image and the file. [F]
Save as...
Save the image in .BMP format.
Reload from file
Reload the current image file [F5].

Copy file...
Copy the current image file on disk. [a-C]
Move file...
Move the current image file on disk. [a-M]
Rename file...
Rename the current image file. [F2, a-R]
Delete file...
Delete/recycle the current image file from the disk. [a-D]
Delete this & show next
Delete/recycle the current file, then show the next file in the folder. [c-X]
Erase file...
Rename & overwrite with random data several times before deleting it. [a-E]
Erase this & show next
Erase the current file, then show the next one in the folder. [s-X]

Copy to clipboard
Copy image to the clipboard (so you can paste it into other apps). [c-C, c-Ins]
Paste from clipboard
Paste an image from the clipboard (replaces current image). [c-V, s-Ins]
Capture a screen shot
Make a copy of the entire screen - minus the Image Eye window.
Capture a window shot
Make a copy a window - select the window to copy within 2 seconds.

View →
Fullscreen mode / Restore window
Toggle between full-screen and normal windowed mode. [F11, a-Enter]
Size window to image
Adjust the window size to the image size. [a-W, F9]
Size image to window
Adjust the image size to the window size. [W, F8]
Restore original size
Restore the image to the size it had when you first opened it. [R, F10]
Trim image to window
Trim the image (cut 'borders') so that it fits inside the window. [T]
Topmost window
Toggle 'topmost', when on, the image will stay on top of other windows. [U]
Freeze window
Freeze or unfreeze the window, see explanation in previous section. [Scroll lock]

Adjust image...
Open the image adjustment dialog. [A, F7]

Rotate image +90°
Rotate the image 90° clockwise. [Cursor keys]
Rotate image -90°
Rotate the image 90° counter-clockwise. [Cursor keys]
Rotate image 180°
Rotate the image 180°.
Mirror image vertically
Mirror the image around the horizontal axis. [c-Cursor keys]
Mirror image horizontally
Mirror the image around the vertical axis. [c-Cursor keys]

Zoom in
Make the image larger. [Cursor keys, Num-Add]
Zoom out
Make the image smaller. [Cursor keys, Num-Sub]
Zoom to factor
Zoom the image to one of the predefined zoom factors. [½ / § / 1 / 2 / 3 / 4]

Use as desktop wallpaper
Set the image as desktop wallpaper (background image).
Send window to back
Put the window behind any other windows on the desktop. [B]

Slideshow

Create slideshow script...
Show slideshow options dialog, then create and run script. [S]
Run in fullscreen mode
Check to run slideshows in full-screen mode.

IndexDisplay a thumbnail index of all images in the folder. [I]

CloseCloses the window / exits the viewer. [Esc, a-X]

In addition to the indicated keyboard short-cuts, you can also use the following mouse and keyboard commands:

Left mouse buttonHold & drag to move window (arrow cursor) or scroll image (scroll cursor).
Right mouse buttonShow application menu.
Mouse wheelScroll vertically.
Mouse wheel + left buttonScroll to display the next/previous image in the folder.
Mouse wheel + shift-key-"-.
Mouse wheel + right buttonScroll to zoom in or out.
Mouse wheel + control-key-"-.
Mouse wheel pressed downDrag window instead of scrolling. [a-Mouse]
Mouse "Browse buttons"Display next/previous file.

Tab/Shift-TabActive next/previous image window [Tab / s-Tab]

Up / Down-cursor keysZoom image in / out. [Cursor keys]
Left / Right-cursor keysRotate counter clock wise / clock wise. [Cursor keys]

Numeric keypad keysScroll the image or moves the whole window in the keys direction. [Key pad]
Alt-keyScroll mode 'override'. [a-Mouse]

Ctrl-Alt-XClose all Image Eye windows. [c-a-X]
