Command line options

    <filename>Display the named image file.
    -indexShow index for the folder <filename>.
    -fullscreenStart in full screen mode.
    -pos=<x>,<y>Show the window at screen coordinates (<x>, <y>).
    -title=<text>Set the window title to <text>.
    -topmostMake window topmost.
    -zoom=<f>Zoom in by factor <f>.
    -onlyoneClose any previous windows.
    -closeClose all Image Eye windows.
    -freezeStart in freezed mode.
