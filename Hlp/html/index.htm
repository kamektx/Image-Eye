Index commands

Selecting Index in the Open image file dialog, or from the Windows Explorer context menu of a folder, opens up a thumbnail index window listing all the images in the current directory.
These thumbnails will be recreated every time you do this - they will not be stored on disk - unless you explicitly give the command to save an index cache-file (for faster loading the next time).

Clicking the right mouse button in an index window brings up a context menu:

View imageOpen the selected image icon (same as left-clicking on icon).
View slideshowView the selected slideshow.
Edit slideshowLaunch NotePad to edit the selected slideshow.
OpenOpen the file with any associated application (for non-image files).
Spawn new...Show the "Open image file" dialog. [N, F4]

File →
Copy file...
Copy the file.
Move file...
Move the file.
Rename file...
Rename the file.
Delete file...
Delete the file.
Erase file...
Rename & overwrite with random data several times before deleting, preventing recovery.

Save index as bitmap...
Saves the index as a .bmp file.

Recurse sub-directoriesFind and display icons for images in all sub-directories.

View →
Directories
Show directories in the index listing.
Non-image files
Show all files types in the index listing.
Freeze window
Prevents window from being resized when you navigate to another folder.

Icon cache →
Recreate
Re-scan folder and recreate thumbnail icons. [F5]
Save cache
Create a disk cache file for the thumbnail icons. [C]
Update cache
Update the disk cache file with new or changed icons. [C]
Remove cache
Delete the disk cache file. [X]

Slideshow →
Slideshow
Create & run slideshow script for all images in the folder. [S]
Create slideshow script...
Show slideshow options dialog, then create and run script. [S]
Run in fullscreen mode
Check to run slideshows in full-screen mode.

OptionsOpen program options dialog. [F6]
HelpOpen this help text. [F1]
CloseClose the index window. [Esc, a-X]

In addition to keyboard shortcuts indicated above, there's also these mouse and keyboard commands:

Left mouse buttonClick icon to open file.
Left button double clickDouble-click background to maximize / restore window. [F11, a-Enter]
Right mouse buttonShow application menu. [Menu]
Mouse wheelScroll vertically.
Mouse wheel pressed downToggle icon selection (use to select multiple icons).
Mouse "Browse back" buttonShow parent folder.

Cursor keysMove among file icons. [Cursor keys]
PgUp / PgDn / SpaceScroll by one page. [PgUp / PgDn, Space]
Home / EndGo to first / last icon. [Home / End]

EnterOpen the file that has keyboard focus. [Enter]
Ctrl + EnterToggle icon selection. [c-Enter]

Ctrl-Alt-XCloses all Image Eye windows. [c-a-X]
