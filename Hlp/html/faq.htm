Frequently Asked Questions

General questions:

QHow much does Image Eye cost?
ANothing for non-commercial use. If you want to use it for commercial purposes, then do contact us for licensing options.

QHow do I uninstall it?
AUse the Windows Control panel's 'Add or remove programs' feature.

QHow can I make Image Eye the default program for opening a certain file type?
AWindows 10/8/7/Vista: Either use the Default Programs function in the Windows control panel (select Set your default programs to set it as the default for all its supported file types, or Associate a file type or protocol with a program to select it on a per file format basis), or right click on a file → Open with → Choose default program...
Windows XP: right click on a file → Open With → Choose Program → Browse → locate Image Eye.exe. Check the box Always use the selected program. Click OK.

QI'd like to be able to add image viewing or image file format conversion capabilities to my own program - can I use your code?
AYes! The underlying ieC++ class library is now Open Source available under the GPL General Public License v3. You can use and modify it freely for free, open source projects. For commercial and/or closed source applications, contact us for licensing options.

QCan I turn off the Recent directory list? I don't want anyone else to see where I've been!
AOpen an image file → right click → Options → Misc, check the box labeled Clear on program exit.

QHow do I delete saved image adjustments for an image?
APress a to open the image adjustments dialog, then Reset. The hidden .iea file containing the adjustment parameters will be deleted and the unmodified original image is displayed.

QHow can I create a Windows shortcut to an Image Eye index for a folder containing images?
AThere's a little trick to accomplish this. Create a shortcut to Image Eye.exe. Then right click on the short-cut, select Properties → Sortcut. In the Target box, after \<path>\Image Eye.exe add a blank space followed by -index followed by another space then the full path to your folder (put it inside quotes if it contains any space).

QHow do I pause a slide show?
AHit the Pause key on the keyboard. Hit any other key to un-pause it.
